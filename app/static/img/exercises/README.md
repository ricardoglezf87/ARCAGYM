# Exercise Image Credits

These local images are mapped from `app/seed/exercise_images.json` and are loaded by
`app.services.exercise_seed_service`.

Most images come from Free Exercise DB:

- Repository: https://github.com/yuhonas/free-exercise-db
- License: Unlicense / public domain dedication

The burpees image comes from Wikimedia Commons:

- File: https://commons.wikimedia.org/wiki/File:CrossFit_Burpee_5_Thrust.jpg
- Credit: `CrossFit Burpee 5 Thrust.jpg` by Taco Fleur
- License: CC BY-SA 4.0

Some exercises use the closest available visual variation when the exact equipment
or movement variant was not available in the source dataset.
